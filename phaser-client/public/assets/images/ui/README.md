# 底部操作栏图片资源

## 从 Figma 导出图片

请从 Figma 设计文件中导出以下图片并放置在此目录：

1. **bottom-platform.png** - 底部平台背景（半圆形，金色边框+浅蓝色表面）
2. **bottom-clouds.png** - 云朵效果（白色云朵+浅蓝色流动效果）
3. **button-bag.png** - 背包按钮（青色束口袋）
4. **button-compass.png** - 装备按钮（金色罗盘）
5. **button-character.png** - 角色按钮（灰色占位符）
6. **button-settings.png** - 设置按钮（灰色占位符）

## 导出步骤

1. 在 Figma 中选中对应的元素
2. 右键选择 "Export" 或使用快捷键
3. 选择 PNG 格式
4. 建议使用 2x 或 3x 分辨率以获得更好的显示效果
5. 将导出的图片重命名为上述文件名并放置在此目录

## 注意事项

- 如果图片未加载，代码会自动使用备用绘制方案（代码绘制）
- 确保图片文件名与代码中的 key 一致
- 建议使用透明背景的 PNG 格式
